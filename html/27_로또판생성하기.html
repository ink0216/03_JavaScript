    같은거 누르면 취소
    아무것도 없다가 로또판 생성 누르면 45 5*9 원 나오고
    생성 버튼 클릭 시 이전 로또판 삭제 -> 새 로또판 생성(초기화)
    최대 6개 선택 
    같은 것 두번 누르면 선택 취소
    
    배치 방법 두가지
    - float : left로 하면 요소를 띄워서 왼쪽정렬 -> 부모 요소 넘어가게 생기면 자동으로 넘어감 
    - display : flex , flex-wrap : wrap, align-content : flex-start(교차축 시작지점으로 정렬) 
        -> 부모 요소를 넘어가게 생기면 자동으로 넘어감 

    선택완료버튼 만들어서 선택한 것 목록 오른쪽에 띄우고 로또판 초기화
    ->완료버튼 누르면 새로 목록 기록되게 만들수도 있다!
