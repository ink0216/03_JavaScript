Script 언어란? 

    한 줄을 읽자마자 바로 실행하는 언어란 
    기본 프로그램의 동작을
    사용자의 요구에 맞게 수행되도록 해주는 용도의 언어

    별도의 소스코드 컴파일(Compile, 2진수로의 번역)을 하지 않고
    Interpreter(인터프리터, 한 줄씩 코드를 읽어서 해석하는 기계, 기능)를 이용해서 직독직해
    장점: 번역하는 데 시간 들지 않음
    단점: 직독직해여서 프로그램 실행 시간이 상대적으로 느림
    <-> Compile 언어 : 인간 언어로 쓴 후 컴퓨터 언어로 번역한 후에 실행 가능

JavaScript란?

    웹 브라우저(크롬, 엣지, 웨일, 파이어폭스 등)에서 
    주로 사용하는 인터프리터 방식의 객체 지향 프로그래밍 언어.
    개발자 도구 콘솔에 쓸 수 있는 것
    
    ECMA(기관)  Script 표준을 따르는 대표적인 웹 기술
    (JS 버전 이름:  ES5, ES6, ES7 ... ) (ECMA Script)
    jQuery
    우리가 배울 것은 ES6기준

JavaScript 코드 작성 방법

  1. 브라우저 개발자 도구 -> Console에 직접 작성하기
  2. HTML 내부에 script 태그를 이용해서 작성하기(Internal 방식, 내부에 작성해서)
  3. HTML 외부에 존재하는 .js 파일을 연결해서 사용하는 방법(External 방식)
  4. 태그에 직접 JS 코드를 작성(inline 방식, 지저분해져서 권장하지 않음)
