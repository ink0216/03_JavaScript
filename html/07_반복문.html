1) for 문

                - 프로그램의 흐름을 제어하는 제어문 중 하나로 
                특정 코드를 계속 반복할 때 사용하는 구문
                -특정 구간을 계속 반복하는 것

                -끝이 정해져 있는 반복에서 주로 사용(반복 횟수 정해진 경우)
                (ex. 몇 번만 반복할 것인지 횟수가 유한하게 정해져 있는 경우)

                [작성법]
                //if(조건식: 결과가 true혹은 false나오는 식){}이었음
                for(초기식;조건식;증감식){
                    조건식이 true인 경우 반복 수행할 코드;
                }

                for문 만났을 때 해석하는 순서
                : 초기식->조건식->중괄호 안의 코드->증감식
                //1~4까지 1회 반복 수, 2~4를 계속 반복함(1234234234234...)
                //그러다가 조건식이 false가 되면 그 즉시 바로 종료


                초기식 : for문을 제어하는 용도의 변수를 선언하는 부분 
                        (10까지만 반복하기 위해 숫자 세는 손의 역할: 초기식)
                조건식 : for문의 반복 여부를 지정하는 식을 작성하는 부분
                        -true일 때 반복하고, false가 될 때 반복을 종료
                        -보통 초기식에 사용된 변수를 이용해서 조건식을 작성한다.
                증감식 : 초기식에 사용된 값을 변화시키는 식(증가시키고 감소시켜서)
                        -> 변환된 초기식에 의해서 조건식의 결과가 달라지게 한다.

    while 문

                    - for, while : ~를 하는 동안, 조건식이 true인 동안
                - for문과 달리 끝이 정확하게 정해져 있지 않은 반복에서 주로 사용
                (목표가 애매모호하게 정해져 있는 경우)
                (ex. 어떤 값이 입력되기 전까지 반복하겠다, )

                [작성법]
                while(조건식){
                    조건식이 true인 경우 반복수행할 코드;
                }
                - 조건식이 true인 경우 내부 코드를 1회 반복 후 
                다시 조건식 검사를 진행
                - 조건식이 false가 나올 때 까지 무한 반복

    메뉴 주문하기
