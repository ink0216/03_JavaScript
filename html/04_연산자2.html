복합 대입 연산자 (+=, -=, *=, /=, %=)

    - 기존 변수에 저장되어있는 값에 
    새롭게 대입 되는 값의 산술 연산 결과를 다시 대입(결과값으로 변수 값이 바뀜)

    -증감 연산자는 1씩만 증가/ 감소하는데
    복합 대입 연산자는 원하는 만큼씩 증가/ 감소 가능
    ex) 
    let temp = 10;
    // temp = 5; 하면 덮어씌워짐
    temp += 5;  == temp = temp + 5;
        (누적(더하기 누적), 차감(빼기 누적), 제곱(나누기 누적) 등등 효과)

논리 연산자(&&, ||, !)

        < 연산자 ( A && B ) >
        - A, B 모두 true일 경우에만 결과로 true를 반환하는 연산자 
        (나머지 경우들은 모두 false로 반환)
        - 문장에서 ~와, 그리고(이고), 이면서(면서), 부터, 까지, 사이
            라는 글귀가 있으면 AND 연산을 수행하면 된다

        - ex) 104는 100 이상의 수 이면서 , 짝수인가?
        - ex) 50은 70 이하의 수 이면서, 4의 배수인가?
        - ex) 3은 1부터 10 사이의 수가 맞는가?
        - 둘다 만족이 돼야 맞는 것이 됨


        < 연산자( A || B )>
        - A, B 모두 false일 경우에만 결과로 false를 반환하는 연산자 
            (나머지 경우는 모두 true)
        - A, B 중 하나라도 true라면 true 반환
        - 문장에서 또는, ~거나(~이거나)
            라는 글귀가 있으면 OR 연산을 수행하면 된다
        
        - ex) 4는 10을 초과하거나, 짝수인가?
        - ex) 50은 0 이하 또는 40 이상인가?
        - 둘 중 하나라도 만족하면 맞는 것이 됨

        <논리 부정(NOT) 연산자(!)>
        -논리 값을 반대로 바꾸는 연산자는

        - ex) let test= true;
                !test // == !true == false로 바뀜

삼항 연산자( 조건식 ? true인 경우 : false인 경우 )

특수 기호 두개 기준으로 세 개의 입력 항 존재 이 위의 연산은 이항 연산자였음 !true : 단항연산자 
        - 조건식의 결과에 따라서 수행되는 코드가 다른 연산자 
        - 조건식의 결과가 true이면 true가 실행되고
        - 조건식의 결과가 false이면 false가 실행됨
        -> 조건식은 항상 결과가 true 또는 false가 반환되도록 작성해야 한다
        예시: 조건식= 1 + 2가 3 이 맞냐?
