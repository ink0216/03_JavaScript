시간 지연 함수

        -window에서 제공하는 시간 관련 함수들은
            모두 비동기로 동작함

비동기(Asynchronous)란?

    -코드 해석(작업)이 작성 순서대로 수행되지 않고
        여러 작업이 동시 수행되는 것
    ->먼저 읽힌 코드가 수행이 완료되기를 기다리지 않고
        바로 다음 코드를 실행하는 것
    앞의 코드 수행 끝난 후에 뒤의 코드 수행하는 것 == 동기 

    [참고] 동기
    -앞의 코드의 수행이 종료 되어야 다음 코드 실행

(window.)setTimeout(함수, 지연시간(ms))

얼마만큼 지연하겠다 
        -매개변수로 주어진 지연시간이 지난 후 함수를 1회 수행하도록 하는 
        window 객체 내장 함수

(window.)setInterval(함수, 지연시간(ms))

        -지연시간이 지날 때 마다 함수를 계속 수행하도록 하는 
        window 객체 내장 함수
        ex) 몇 초마다 수행해라 (몇 초에 한 번씩 수행해라)
