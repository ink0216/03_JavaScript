요소 생성 : document.createElement("태그명")

        화면에 없던 요소를 만드는 것!
        1) DOM 객체에 요소를 하나 생성하는 것
            -> body 태그 안에 만들어야 화면에 나오는데, 
                document에 만들어지므로 body 태그 밖에 생성됨->만들어도 화면에 안보임
                ->body 태그 내부의 필요한 위치에 끼워넣을 수 있다
        2) DOM 객체에 요소를 하나 생성하면 요소는 생성되어 있지만
            HTML(body) 객체 내부에 생성되는 것이 아닌
            외부에 생성되어 안보임(존재는 하지만 화면에 보이지 않아ㅠㅠ) 
        3) 노드/요소 추가 구문(append(), after(), ...)을 이용해서
            원하는 위치에 배치하면 보이기 시작함
컨테이너 안에 버튼 누를 때마다 컨테이너의 마지막 자식으로 네모가 생성되도록 만들어보기! innerHTML과의 차이점도 보기! 
        * innerHTML 요소 생성 원리
        
        ex) A.innerHTML = "...";     (...에는 태그 포함)
    
        1) 특정 요소(A) 내부의 모든 내용(코드) 삭제
    
        2) 새로운 내용(...)을 A에 내용으로 대입해서 덮어씌움
    
        3) HTMLParser를 이용해서 ...에 포함된 태그를 해석
