조건문(condition)

        프로그래밍 코드가 해석되는 순서를 
        
        제어하는 제어문(건너뛰는 등, 좌->우, 상->하 방향으로 코드 해석하는 일반적 흐름을 깨는 것들)의 한 종류로

        작성된 조건식에 따라서 수행되는 코드를 변경하는 구문

if(만약에 ~하다면) 문

        
            if(조건식){
                조건식이 true인 경우 실행할 코드;
            }
        
        *조건식: 결과가 true, false인 식(연산)
        *조건문: 조건식의 결과에 따라 실행을 제어하는 구문(if, switch)

입력받은 수가 양수인지 아닌지 검사

if( ~ 라면 ~해라) - else( ~ 아니라면 ~해라)문

        
            if(조건식){
                조건식이 true인 경우 수행할 코드 적기
            }
            else{
                
                조건식이 false인 경우 수행할 코드 적기
                //아닌 경우의 조건식은 쓸 필요 없음
            }
        
        * else 구문에는 조건식을 작성하는 ()가 없다!!!!!

0 ~ 100 사이 난수(랜덤인 수)를 발생시켜 홀/짝 판별하기

근데 0은 홀수도 짝수도 아닌 것이 문제!

if(이거라면) - else if(아니면 이거라면)(else if는 안써도 되고 여러 번 쓸 수도 있음) - else(모두 아니면)

        
            if(조건식){
                조건식이 true인 경우에 실행할 코드 작성된
            }

            else if(조건식{
                앞선 if의 조건이 false이면서,
                else if의 조건식이 true인 경우에 실행할 코드 작성
            })

            else{
                모든 if, else if의 조건식이 모두 false인 경우에 실행할 코드 작성
                만약 if나 else if 구문 중 true가 되는 조건식이 나온다면 그 뒤의 코드는 실행되지 않는다.
            }
        
        * else if 구문은 원하는 만큼 작성 가능하다!!! 
    
        * 앞선 if/else if문 조건이 false인 경우
            이후 else if에서 해당 범위는 제외된다

-3 ~ 3 사이 난수를 발생시켜 양수, 음수, 0 판별하기

어린이, 청소년, 성인 구분하기

switch구문(switch, case, default 구문)

    switch문 (리모컨처럼 이런 케이스면 이렇게, 저런 케이스면 저렇게 )
    - 식 하나의 결과로 많은 경우의 수를 처리할 때 사용하는 조건문
    -> 식의 결과에 맞는 case 구문이 수행됨
     
        switch(식){ // 식의 결과는 정수 또는 문자열
    
        case 결과1 : 결과1의 수행 코드; break;
        case 결과2 : 결과2의 수행 코드; break;
        case 결과3 : 결과3의 수행 코드; break;
        ...
        
        default : 모든 case가 만족하지 못할 때 수행할 코드
        }
